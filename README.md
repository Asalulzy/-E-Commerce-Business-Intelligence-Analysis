# 🛒 E-Commerce Business Intelligence Analysis

An end-to-end Business Intelligence project analyzing the Brazilian Olist E-Commerce dataset to uncover actionable business insights from customer transactions, product sales, payment behavior, delivery performance, and customer satisfaction.

This project demonstrates a complete data analytics workflow, starting from multi-table data integration and cleaning, followed by exploratory data analysis (EDA), feature engineering, business KPI analysis, and insight generation to support data-driven business decision making.

---

## 📌 Project Objectives

This project aims to answer several key business questions, including:

* How has sales performance evolved over time?
* Which product categories contribute the highest revenue?
* What purchasing behaviors do customers exhibit?
* How efficient is the order fulfillment and delivery process?
* Which payment methods are most frequently used?
* What factors influence customer satisfaction and review scores?
* Which sellers generate the best business performance?

---

## 📊 Dataset

The analysis uses the **Olist Brazilian E-Commerce Public Dataset**, consisting of multiple relational tables representing different aspects of an e-commerce platform, including:

* Customers
* Orders
* Order Items
* Products
* Sellers
* Payments
* Reviews
* Product Category Translation

These datasets were integrated into a unified analytical dataset to enable comprehensive business analysis.
